# PEGO

Personal Executive Governance OS.

This private repository is the first PEGO instance: a delegated life-governance system where the human supplies values, goals, constraints, telemetry, and concerns, while agents produce directives, plans, tradeoff analyses, and escalations.

PEGO is not a helper, habit tracker, or quantified-self dashboard. The operating premise is stronger: the system is allowed to make decisions for the human inside an explicit governance model.

## Repository Boundary

- `pego/` contains reusable framework material that may eventually become open source or productized.
- `private/` contains this person's private PEGO instance: real goals, constraints, telemetry, directives, and life details.
- `ops/` contains scripts, integrations, automation, and local operating machinery.
- `decisions/` contains durable decision records about PEGO's architecture and governance.

Do not place private life details, secrets, credentials, real financial data, health records, or personal journals in `pego/`.
