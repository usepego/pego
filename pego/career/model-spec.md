# PEGO Career Model Spec

Reusable specification for PEGO career and work strategy.

This file must not contain private employer details or personal compensation data.

## Purpose

The PEGO career model evaluates how work should serve the person's life aims, financial freedom, autonomy, skill growth, reputation, and household constraints.

## Inputs

- Current role.
- Compensation.
- Benefits.
- Equity or ownership.
- Flexibility.
- Work-from-home status.
- Skill growth.
- Reputation growth.
- Network growth.
- Energy cost.
- Meaning and interest.
- Alignment with desired future.
- Independent-work potential.
- Household constraints.

## Outputs

- Stay/bridge/exit recommendation.
- Career capital score.
- Upside path map.
- Exit-readiness status.
- Negotiation opportunities.
- Portfolio/proof-of-work plan.
- Skill acquisition path.
- Network plan.

## Strategy States

- Stay and compound.
- Stay as income base.
- Shape role for upside.
- Build bridge.
- Prepare exit.
- Exit.

## Review Cadence

- Weekly: active work leverage and skill extraction.
- Monthly: role and upside review.
- Quarterly: stay/bridge/exit review.
- Event-driven: compensation, equity, leadership, role, or household changes.
