# Active Operating Brief Template

Use this as the local private entry point for the current PEGO operating state.

The active operating brief should tell agents which files and directives are currently in force, what the next test is, and what governance limits apply.

## Current Operating Mode

What mode is PEGO in right now?

Examples:

- Framework setup.
- First practical use.
- Daily execution.
- Weekly review.
- Monthly strategy review.
- Escalation review.

## Active Objective

What is PEGO trying to prove or accomplish now?

## Active Private Files

### Daily

Current daily directive, synthesized day plan, or morning prompt.

### Weekly

Current weekly operating plan.

### Monthly

Current monthly strategy review or strategy priorities.

### Directive Candidates

Current candidate list.

### Operating Register

Current register for upcoming events, recurring annoyances, supply gaps, prep needs, home watchlist, strategic dependencies, and concrete concerns.

### Domain Files

Active files by domain.

## Current Test

What near-term test is PEGO running?

## Minimum Success

What counts as success if the full plan does not happen?

## Current Governance Notes

- Current authority level.
- Explicitly approved actions.
- Explicitly forbidden actions.
- Escalations required.
- Privacy constraints.

## Next Review

When should PEGO review this brief?
