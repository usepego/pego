# Agent Recommendation Review

Use this template to evaluate whether a PEGO agent's recommendation improved
the eventual directive and outcome.

## Date

Date.

## Reviewed Agent

Agent name and domain.

## Source Recommendation

Path or identifier of the recommendation being reviewed.

## Related Council Decision

Path or identifier of the council decision, if any.

## Related Directive Outcome

Path or identifier of the directive outcome, if any.

## Recommendation Summary

What the agent recommended or requested.

## Outcome Summary

What happened after the recommendation was adopted, revised, deferred, blocked,
or rejected.

## Fit Assessment

Did the recommendation fit time, energy, location, protected time, authority,
privacy, and current constraints?

## Friction Prediction

Did the agent correctly identify the likely friction, blocker, or behavior
loop?

## Information Request Quality

If the agent requested more information, was the requested information
decision-grade and worth asking the human?

## Stress Impact

Did the recommendation reduce, preserve, or increase human anxiety, decision
load, or stress?

## Evidence Quality Review

Was the agent's evidence quality accurate, overstated, understated, or missing?

## Dissent Quality

Did the agent preserve a useful opposing view or handoff?

## Review Outcome

Improve / Keep / Downgrade / Quarantine / Escalate.

## Future Adjustment

What should change in future recommendations from this agent?

## Durable Memory Candidate

What, if anything, should become a context update, operating-register item,
agent assumption, or governance note?

## Next Review

When or under what condition this review should be revisited.
