# Anticipation Scan Template

Use this template to identify future friction before it becomes urgent.

## Date

Date.

## Horizon

Today / 7 days / 14 days / 30 days / 90 days / Event-specific.

## Trigger

Calendar event / Seasonal condition / Recurring annoyance / Goal dependency / Supply gap / Human concern / Agent observation / Other.

## Domain

Environment / Event / Food and health / Strategy / Finance and admin / Relationship / Other.

## Known Facts

Decision-grade facts already known.

## Unknowns

Facts PEGO needs before directing action.

## Targeted Question

One concrete operational question, if needed.

## Candidate Directive

The smallest useful action that would prevent friction.

## Lead Time

How early this must happen to avoid scrambling.

## Dependencies

Supplies, weather, schedule, money, another person, documents, tools, or external availability.

## Priority

Low / Medium / High.

## Consequence of Deferral

What happens if this waits?

## Governance Status

Authority level, privacy status, stakeholder impact, spending impact, and review requirement.

## Next Step

Ask targeted question / Add directive candidate / Schedule / Defer / Escalate / Update context.
