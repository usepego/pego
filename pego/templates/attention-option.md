# Attention Option Template

Use this template when PEGO evaluates whether an event, media item, or leisure activity deserves attention.

## Date

Date and window.

## Event

Name of event or media item.

## Source

Calendar / Sports schedule / News / Social input / Streaming service / Manual report / Agent observation / Other.

## Event Type

Sports / News / World event / Culture / Learning / Entertainment / Social / Rest / Other.

## Live Value

High / Medium / Low / Unknown.

## Personal Importance

High / Medium / Low / Unknown.

## Recovery Value

High / Medium / Low / Unknown.

## Social Or Cultural Value

High / Medium / Low / Unknown.

## Opportunity Cost

What else this attention could be used for.

## Multitask Compatibility

Full attention required / Compatible with low-cognitive work / Background only / Not compatible / Unknown.

## Time Window

When this can be watched, checked, summarized, or skipped.

## Best Alternative

Highlights, summary, score check, replay, article, or no follow-up.

## Risk

Time sink, sleep disruption, avoidance, anxiety, distraction, or other risk.

## Recommendation

Watch live / Multitask live / Highlights later / Score only / Defer / Skip / Escalate.

## Stop Condition

What should cause PEGO to revise the recommendation?
