# Behavior Loop Template

Use this template when PEGO identifies a recurring environment-driven pattern
that either supports or works against an active strategy.

Structured implementations should emit `pego/schemas/behavior-loop.schema.json`.

## Loop

Short name.

## Domain

Health / Finance / Career / Venture / Home and Environment / Relationships /
Exploration / Communications / Happiness / Operations.

## Trigger

What location, time, emotion, social context, app, store, route, event, or
physical condition tends to start the loop?

## Routine

What does the human tend to do?

## Reward or Relief

What immediate reward, relief, convenience, identity reinforcement, stimulation,
or avoidance does the routine provide?

## Strategic Effect

Supports strategy / Works against strategy / Mixed / Unknown.

## Evidence

Human report / Repeated outcome / Telemetry / Agent inference / Unknown.

## Replacement Frame

What environment, route, prompt, object, default, companion, timing, or
constraint should replace the old loop?

## Disruption Directive Candidate

What small directive should PEGO issue the next time the trigger appears?

## Guardrails

What should PEGO explicitly prohibit, avoid, limit, or route around?

## Authority Level

Level 0 / Level 1 / Level 2 / Level 3 / Level 4.

## Review Rule

When should PEGO review whether the disruption worked?
