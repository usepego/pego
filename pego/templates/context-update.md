# Context Update Template

Use this template to convert new information into private PEGO operating memory.

Use `pego/templates/memory-lifecycle-review.md` when deciding whether the update
should become working memory, durable private memory, quarantined memory,
expired memory, or rejected.

## Date

Date.

## Source

Conversation / Outcome / Telemetry / Document / Agent observation / External input / Other.

## Raw Observation

What was learned?

Keep raw private content in private files only.

## Update Class

Fact / Preference / Constraint / Goal / Strategy / Pattern / Tone rule / Voice rule / Taste signal / Influence / Public positioning / Governance rule.

## Evidence Strength

Direct statement / Repeated statement / Observed behavior / Directive outcome / Telemetry / Model output / Professional input / Agent inference / Speculation.

## Stability

Stable / Current but changeable / Provisional / Needs confirmation.

## Destination File

Private file to update.

## Proposed Update

The exact change to make.

## Affected Agents

Which agents should use this update?

## Governance Impact

Authority, privacy, protected time, stakeholder, or escalation implications.

## Action

Record only / Update destination / Request confirmation / Escalate / Reject.

## Review Date

When should this update be reviewed?
