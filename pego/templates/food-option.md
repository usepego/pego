# Food Option Template

Use this template when PEGO evaluates a possible meal, grocery item, restaurant item, snack, or prepared food option.

## Date

Date.

## Source

Manual report / Home inventory / Grocery source / Restaurant source / Menu source / Nutrition database / Map service / Delivery app / Agent estimate / Other.

## Source Confidence

High / Medium / Low / Unknown.

## Location Type

Home / Grocery / Restaurant / Delivery / Workplace / Travel / Other.

## Provider

Store, restaurant, household source, or other provider.

## Item

Food item or meal.

## Components

Known components, ingredients, or portion notes.

## Availability

When and where this is available.

## Cost Estimate

Estimated cost.

## Time And Friction

Prep time, travel time, ordering friction, weather, cleanup, or social friction.

## Nutrition Estimate

Calories, protein, fiber, sugar, sodium, and other relevant nutrients.

## Goal Fit

How this supports or conflicts with the current health goal.

## Enjoyment Fit

How likely the person is to enjoy and accept this option.

## Satiety Estimate

How likely this is to prevent hunger or snacking later.

## Tradeoffs

What is gained and what is lost by choosing this.

## Stop Condition

What should cause PEGO not to recommend this option?
