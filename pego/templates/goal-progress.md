# Goal Progress

Use this template when PEGO summarizes progress toward a domain goal from
state signals, outcomes, telemetry, financial activity summaries, or agent
observations.

Structured runtimes should emit `pego/schemas/goal-progress.schema.json`.

Date.

## Domain

Exploration / Happiness / Operations / Governance / Communications.

## Goal ID

Stable local identifier for this goal.

## Source Signals
