# Information Value Assessment

Use this template before PEGO asks the human for more information during active
operation.

The purpose is to ask fewer, better questions.

## Date

Date.

## Decision Frame

The directive, council decision, recommendation, or escalation that may require
more information.

## Proposed Question

The exact question PEGO proposes to ask the human.

## Requesting Agent

Agent, council, or governance layer requesting the information.

## Needed By

Which agents or decisions would use the answer.

## Current Known State

What PEGO already knows that is relevant.

Do not include private facts in reusable public examples.

## Missing Information

The specific missing fact, constraint, preference, status, or evidence.

## Decision Impact

How the answer would change the next directive, deferral, escalation, or stop
condition.

## Inference Option

Can PEGO infer safely instead of asking? If yes, state the assumption.

## Human Burden

Time, cognitive load, emotional load, privacy sensitivity, or interruption cost.

## Risk of Not Asking

What could go wrong if PEGO proceeds without the answer?

## Risk of Asking

What could go wrong if PEGO asks now?

## Assessment Outcome

Ask now / Infer and proceed / Defer question / Escalate / Block.

## Final User-Facing Question

If asking, write the smallest question the human should see.

## Fallback If Unanswered

What PEGO should do if the human does not answer.

## Review

How PEGO should later evaluate whether asking this question was useful.
