# Memory Lifecycle Review Template

Use this template when PEGO reviews session, outcome, telemetry, tool, or
context evidence for promotion, quarantine, expiry, correction, or rejection.

Structured runtimes should emit
`pego/schemas/memory-lifecycle-review.schema.json`.

## Review

Short name.

## Date

Date.

## Source Artifacts

Session logs, outcome reviews, context updates, tool outputs, behavior loops,
agent messages, council decisions, or other protected sources.

## Candidate Memory

What PEGO may remember.

## Memory Tier

Session / Working / Durable private / Quarantined / Expired.

## Proposed Action

Record only / Promote to working memory / Promote to durable private memory /
Quarantine / Expire / Correct or supersede / Reject.

## Evidence Strength

Direct statement / Repeated statement / Observed behavior / Directive outcome /
Telemetry / Model output / Professional input / Agent inference / Speculation.

## Stability

Stable / Current but changeable / Provisional / Needs confirmation / Stale.

## Destination

Protected private destination file or memory area.

## Affected Agents

Agents that should use or ignore this memory.

## Governance Impact

Authority, privacy, protected-time, stakeholder, legal, medical, tax, financial,
career, housing, or relationship implications.

## Expiry or Review Rule

When should this memory be reviewed, expired, or corrected?

## Decision

Apply / Defer / Quarantine / Expire / Correct / Reject.

## Reasons

Why this decision was made.
